Basic HTML/CSS

What is HTML?

HTML stands for Hypertext Markup Language.

This is what you would use to as the skeleton of a webpage structure. The structure of a webpage is build with blocks we call html elements. We call these elements tags.

<tag_name> - this is the start a tag

</tag_name> - this is the end of a tag

There are special cases where tags won't need the forward slash '/' in the end tag

Tag Examples

  • <html> container for all the html elements </html>
  • <head> tag containing meta (abstract) info about the document </head>
  • <p> defines a paragraph </p>

Creating Your Own HTML Page

Now that we know a bit about HTML and tags, lets try to create our own html page!

The first step to creating an html page is having a text editor workspace to type your code into. There are many options you can use: textEdit, Atom, Sublime, etc. Once you have one opened, the first thing you should do is type the <!doctype html> declaration! This will be the first thing your browser will read from your document. The declaration will let your browser know what version of html your page is written in.

Within our html tag, we will be placing all of our other html elements, these include paragraphs, titles, images, etc. For now, let's make a simple website that will share a bit about ourselves.

Within our html tag, let's create a new tag:<head></head> Remember that this tag will contain abstract information about our page. For now, we will be placing the title of our page in this tag. Our page title will be encased in another tag: <title></title>. Whatever we place inside our title tag will be displayed on the tab your website is opened on. Put your name inside the title tag.

Now let's open our website on a web browser to see how it looks so far. Follow these steps:

What is CSS?

CSS stands for Cascading Style Sheets. Basically, this will describe how to display your html on your browser. Syntax for CSS is different for html. Instead of tags, CSS uses rule-sets. These are the basic components of a rule set:

selectors are used to identify certain things in your html document you want to style
